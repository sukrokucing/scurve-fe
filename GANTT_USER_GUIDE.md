# Gantt View User Guide

## Quick Reference

### Editing Tasks in Gantt View

**Method**: Double-click to edit

You can edit tasks in the Gantt view by **double-clicking** on:
1. The task bar in the timeline
2. The task name in the table

This will open the streamlined edit dialog.

#### Task Dependencies
- **Add Dependency**: Use the Predecessors dropdown to select a task
- **Remove Dependency**: Click the X button on the dependency badge
